JavaForce
JavaForce (JF) is a Java library extending the capabilities of Java.

JavaForce provides:
  • VoIP SIP and RTSP
  • Native code for FFMPEG, OpenGL, OpenCL, Camera.
  • Communicate with PLCs (Siemens, AllenBradley, ModBus)
  • Native Launcher (GUI, Console, Services)
  • Supports Windows, Linux/FreeBSD and MacOS
  • and much more...

JavaForce apps:
  • jfKVM Virtual Machine Management System (libvirt)
  • jfBackupTape Backup System
  • jfPhone VoIP SIP SoftPhone
  • jfPBX VoIP SIP PBX
  • jfBroadcast VoIP SIP AutoDialer
  • PLC Data Logger
  • PLC Data Logger Plus
  • Telnet client
  • FTP Client
  • Torrent client/tracker/maker
  • Passwords Manager
  • Text Editor
  • Paint (Image Editor)
  • Video Creator
  • Audio Editor
  • Music Editor
  • jfDVR RTSP/H.264/H.265 Video Recording
  • Media Player
  • Record Desktop
  • Record Camera
  • jfMonitor Network Monitoring System
  • jfNetBoot PXE Network Booting Thin Client Server
  • SSH Server
  • Proxy Server
  • DNS Server
  • SOCKS4/5 Server
  • DHCP Server
  • STUN/TURN Server
  • Network Tools
  • VNC Client
  • and many more...
