JavaForce
JavaForce (JF) is a Java library extending the capabilities of Java.

JavaForce provides:
  • VoIP SIP and RTSP
  • JNI bindings for FFMPEG, OpenGL, Camera, Raspberry PI GPIO, NI-DAQmx
  • Communicate with PLCs (Siemens, AllenBradley, ModBus, National Instruments)
  • Native Wrapper (with Windows Service support)
  • OpenGL framework (.3ds and .blend files supported)
  • Supports Windows, Linux/FreeBSD and MacOS
  • and much more...

JavaForce apps:
  • jfDVR (RTSP/H.264 Video Recording)
  • jfPhone (VoIP SIP SoftPhone)
  • jfPBX (VoIP SIP PBX)
  • jfBroadcast (VoIP SIP AutoDialer)
  • PLC Data Logger
  • PLC Data Logger Plus
  • Telnet client
  • FTP Client
  • Torrent client/tracker/maker
  • Passwords Manager
  • Paint
  • Video Creator
  • Audio Editor
  • Music Editor
  • Media Player
  • Record Desktop
  • Record Camera
  • Proxy Server
  • DNS Server
  • SOCKS4 Server
  • DHCP Server
  • STUN/TURN Server
  • Text Editor
  • Network Tools
  • Remote Desktop Server(RDP)
  • VNC Client
  • Enterprise Tape Backup System
  • and many more...
