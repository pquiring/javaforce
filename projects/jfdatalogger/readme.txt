jfDataLogger
============

jfDataLogger can log data from PLC controllers.

Supports:
 - Siemens S7-300/400 PLCs
 - AllenBradley rsLogix5000 series
 - ModBus IP protocol
 - National Instruments DAQmx systems
 - Microphones (dB level)

Has the option to save to a csv file.

Website: http://jfdatalogger.sourceforge.net
