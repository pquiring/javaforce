CephFS Storage:
===============

Desc:
-----
Replicated file system for storage pools (Red Hat) (vSAN).

Requirements:
-------------

Three or more servers in a cluster.
Each server has available partitions.

Instructions:
-------------

First go to the Cluster screen and perform a 'Ceph Setup'.

Add a new Storage Pool, select CephFS on each server.

CephFS is always mounted to /volumes/cephfs.

Ceph CLI:
---------

View Status:
  ceph status

View available disks:
  ceph orch device ls

Zap disk:
  ceph orch device zap {host} {device} --force

Add all available disks:
  ceph orch apply osd --all-available-devices
