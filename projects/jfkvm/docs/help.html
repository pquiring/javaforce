jfKVM

jfKVM:
=====

Virtual Machine Management System.
Powered by libvirt:


Desc:
-----

jfKVM is a Virtual Machine Management System (libvirt)
Designed to be a drop-in replacement for VMWare ESXi hypervisor.

Please note this is minimally tested software and not recommended for production environments!

Features supported:
 - Linux and Windows guests
 - Disks : vmdk, qcow2, iso (thick and thin provisioning)
 - Networking : bridge, guests on VLANs (openvswitch)
 - Storage Pools : Local Disk Partition, NFS, iSCSI
 - import vmware machines
 - live/offline compute migration
 - offline data migration/clone
 - autostart machines
 - iSCSI requires manual setup (see Storage below for more info)

Not supported:
 - VMFS storage pools
 - vmdk file split into multiple 2GB extents
