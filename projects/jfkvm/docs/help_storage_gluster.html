Gluster Storage:
==============

Desc:
-----
Replicated Disk File system for storage pools (Red Hat) (vSAN).

Requirements:
-------------

Two or more servers in a cluster.
Each server has an available partition of the same size.

Instructions:
-------------

First go to the Cluster screen and perform a 'Gluster Probe' for each server from each server.

Add a new Storage Pool, select Gluster and an available partition on each server.

Format the pool on each server (xfs).

Start the pool on each server.

From one server perform a 'Gluster Create Volume'.

From one server perform a 'Gluster Start Pool'.

Now you should be able to mount the pool on all servers.
