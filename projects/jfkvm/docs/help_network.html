jfKVM

Networking:
===========

Networks
--------

This is where you can configure VLANs that guest VMs can connect to.


Virtual Switch (bridge)
-----------------------

This is a bridge that all virtual machines and server virtual interfaces connect to.
The bridge should be connected to a secondary physical network interface (not the physical management interface).

Server Virtual NIC
------------------

These allow you to create addition virtual network interfaces on your server on different VLANs.
For example, your server may need to access NFS or iSCSI targets on a different VLAN from the management network.

Physical NICs
-------------

Displays the status of all physical NICs.
