jfKVM

API: [since 5.0]
====

jfKVM includes a web-based https API.

General format:

https://server/api/command?arg1=value1&argN=valueN&...

Commands:

  snapshot_create
    token = server token
    vm = virtual machine name
    name = snapshot name
    desc = snapshot desc

  snapshot_delete
    token = server token
    vm = virtual machine name
    name = snapshot name
