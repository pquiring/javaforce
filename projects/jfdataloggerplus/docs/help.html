Host/Tag format examples:
  • Siemens : Host=IP Address, Tag Examples=DB0.DBW1, I2.3, IW4, M5.6
    • Peripheral Input not supported (create logic to copy them to DB or MB)
  • AllenBradley : Host=IP Address, Tags=main controller tags only (task tags not supported).
  • ModBus : Host=IP Address, Tag Examples=DI1
  • National Instruments : the host specifies the controller and port options (the tag is not used).
  • NI Examples:
    • AnalogInput : Host=cDAQ9188-189E9F4Mod6/ai0 DataType=float64
    • DigitalInput : Host=cDAQ9188-189E9F4Mod8/port0/line0 DataType=int8
    • CounterInput : Host=cDAQ9188-189E9F4Mod1/ctr0/pfi0 DataType=float64
