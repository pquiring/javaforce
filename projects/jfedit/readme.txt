jfEdit
======

A multi-tabbed text editor.

Press F1 for help.
Press F2 for options.

Note:
If you notice the cursor leaving a trail you have a bad 3D video card (ie: nVidia 8400M GS).
  Use "-Dsun.java2d.d3d=false" java option to disable 3D acceleration in Java to avoid the annoying cursor trails.

Author : Peter Quiring
